---
title: Typing
description: How textseek keeps results typed
---

`extract_one`, `extract_first`, and `extract_all` are overloaded per spec type,
so your IDE narrows the result automatically:

```python
from textseek import TextSeek, Extract

seek = TextSeek(text)
code = seek.extract_one(Extract.code(length=6))   # CodeResult
link = seek.extract_one(Extract.link())            # LinkResult
link.query_params["token"][0]                      # fully typed
```

All public methods are annotated, results are frozen dataclasses, and `Any` is
avoided in the public API.
