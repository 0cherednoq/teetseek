---
title: Поиск по регулярным выражениям
description: Произвольное извлечение
---

```python
from textseek import TextSeek, Extract

result = TextSeek("Order ID: AB-9921").extract_one(
    Extract.regex(r"Order ID:\s*([A-Z0-9\-]+)")
)
print(result.value)  # AB-9921 (первая группа, так как в шаблоне есть группы)
result.groups        # кортеж групп
result.groupdict     # именованные группы
result.start
result.end
```

Если в шаблоне есть группы, `value` содержит первую группу; иначе — всё совпадение целиком.
Некорректный шаблон вызывает `RegexExtractError` в момент вызова `Extract.regex`.
