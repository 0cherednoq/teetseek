---
title: TextSeek
description: Основной объект-точка входа
---

`TextSeek(text)` оборачивает строку и предоставляет четыре метода. Каждый принимает
спецификацию `Extract` и передаёт управление соответствующему экстрактору.

| Метод | Возвращает |
|---|---|
| `extract_one` | один результат или вызывает `ExtractNotFoundError` |
| `extract_first` | первый результат или `None` |
| `extract_all` | список результатов (возможно, пустой) |
| `contains` | `bool` |

```python
seek = TextSeek("Your code is 123456")
seek.extract_one(Extract.code(length=6))
seek.extract_first(Extract.link())
seek.extract_all(Extract.regex(r"\d+"))
seek.contains(Extract.code(length=6))
```

<Note>
`extract_all` возвращает совпадения в порядке исходного текста, **кроме**
`Extract.code(near=...)`, где результаты ранжируются по близости к фразе (сначала
коды после фразы). `extract_one`/`extract_first` затем возвращают лучшее по рангу
совпадение.
</Note>
