---
title: Спецификации извлечения
description: Как создаются правила поиска
---

Внутренние классы спецификаций никогда не создаются напрямую. Используйте фабрику `Extract`:

```python
Extract.code(...)   # confirmation codes
Extract.link(...)   # links
Extract.regex(...)  # arbitrary regex
```

Каждый вызов возвращает неизменяемый объект-спецификацию, который проходит валидацию
в момент создания. Некорректные настройки вызывают исключение немедленно
(например, `InvalidCodeSpecError`), ещё до запуска поиска.
