## Установка

```bash
pip install textseek
```

Требуется Python 3.12+. Без зависимостей во время выполнения.

## Извлечение кода из письма с подтверждением

```python
from textseek import TextSeek, Extract

email = """\
From: Acme <no-reply@acme.com>
Subject: Your verification code

Hi Alex,

Your verification code is 482913. It expires in 10 minutes.
If you didn't request this, you can ignore this email.
"""

code = TextSeek(email).extract_one(
    Extract.code(length=6, digits=True, near="verification code")
)
print(code.value)  # 482913
```

`near="verification code"` привязывает поиск к фразе, поэтому случайное число в
другом месте письма не будет выбрано по ошибке.

## Извлечение ссылки из письма с magic-ссылкой

```python
from textseek import TextSeek, Extract

email = """\
From: Acme <no-reply@acme.com>
Subject: Sign in to Acme

Click the link below to sign in:
https://acme.com/auth/magic?token=abc123&email=alex@acme.com

This link expires in 15 minutes.
"""
```

**Обычный способ** — фильтр по домену и чтение структурированных частей:

```python
link = TextSeek(email).extract_one(Extract.link(domain="acme.com"))
print(link.path)                      # /auth/magic
print(link.query_params["token"][0])  # abc123
```

**По примеру** — передайте образец URL и находите ссылки такой же формы (его
схема, хост, путь и ключи query становятся фильтром; значения игнорируются):

```python
link = TextSeek(email).extract_one(
    Extract.link(sample="https://acme.com/auth/magic?token=TEMPLATE&email=user@acme.com")
)
print(link.value)  # https://acme.com/auth/magic?token=abc123&email=alex@acme.com
```
