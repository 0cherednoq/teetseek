---
title: TextSeek
description: Справочник API для класса TextSeek
---

`TextSeek(text: str)` оборачивает строку. Каждый метод принимает спецификацию
`Extract` и перегружен (`@overload`) по типу спецификации, поэтому тип результата
сужается автоматически.

## `extract_one(spec) -> ExtractResult`

Лучшее единственное совпадение. Возвращает `CodeResult` / `LinkResult` /
`RegexResult` в зависимости от спецификации.

Вызывает `ExtractNotFoundError`, если совпадений нет; `InvalidExtractSpecError`,
если тип спецификации не распознан.

## `extract_first(spec) -> ExtractResult | None`

Первое совпадение или `None`. При `near` «первое» означает лучшее по рангу, а не
по порядку в тексте.

## `extract_all(spec) -> list[ExtractResult]`

Все совпадения (пустой список, если нет). Для `code` с `near` результаты
упорядочены по близости к фразе (сначала после фразы); иначе — в порядке
исходного текста.

## `contains(spec) -> bool`

`True`, если есть хотя бы одно совпадение. Эквивалентно
`extract_first(spec) is not None`.
