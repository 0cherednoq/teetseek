---
title: Результаты
description: Датаклассы результатов
---

Все результаты — это `@dataclass(slots=True, frozen=True)`.

## CodeResult

`value: str`, `start: int`, `end: int`

## LinkResult

`value`, `start`, `end`, `scheme`, `domain`, `host`, `port: int | None`,
`path`, `query`, `fragment`, `query_params: dict[str, list[str]]`

## RegexResult

`value`, `start`, `end`, `groups: tuple[str, ...]`, `groupdict: dict[str, str]`
