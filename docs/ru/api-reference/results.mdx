Все результаты — это `@dataclass(slots=True, frozen=True)` с общими полями
`value`, `start`, `end`.

| Поле | Тип | Значение |
|---|---|---|
| `value` | `str` | Найденная подстрока. Для ссылок хвостовая пунктуация `.,;:!?)]}>"'` отрезается. |
| `start` | `int` | Начальный индекс в исходном тексте. |
| `end` | `int` | Конечный индекс (не включая) в исходном тексте. |

## CodeResult

Ничего не добавляет сверх `value`, `start`, `end`.

## LinkResult

| Поле | Тип | Примечания |
|---|---|---|
| `scheme` | `str` | Например, `"https"`. |
| `domain` | `str` | Хост. **Псевдоним `host`** — то же значение. |
| `host` | `str` | Хост. |
| `port` | `int \| None` | `None`, если отсутствует или некорректен. |
| `path` | `str` | Например, `"/auth/magic"`. |
| `query` | `str` | Сырая строка query. |
| `fragment` | `str` | Часть после `#`. |
| `query_params` | `dict[str, list[str]]` | Разобранный query; пустые значения сохраняются (`?a=` → `{"a": [""]}`). |

## RegexResult

| Поле | Тип | Примечания |
|---|---|---|
| `groups` | `tuple[str, ...]` | Позиционные группы. |
| `groupdict` | `dict[str, str]` | Именованные группы. |

`value` — это первая группа, если в шаблоне есть группы, иначе всё совпадение.
