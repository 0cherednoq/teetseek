title: Введение

---

`textseek` — это Python-библиотека без зависимостей для поиска и извлечения
структурированных данных из простого текста: кодов подтверждения, ссылок и
произвольных совпадений регулярных выражений.

## Что умеет

- Находить числовые, буквенно-цифровые и символьные коды подтверждения
- Извлекать ссылки `http`/`https` как полноценные объекты со свойствами `scheme`, `domain`, `path`, `query_params`
- Фильтровать ссылки по домену, пути, параметрам запроса или примеру ссылки
- Выполнять произвольное извлечение с помощью регулярных выражений с типизированными результатами

## Чего НЕ делает

`textseek` — это **не** библиотека для работы с почтой или HTTP. В ней нет IMAP, Gmail/Outlook
API, SMTP, HTTP-клиента, OAuth2, базы данных или парсеров для конкретных провайдеров. Получение
текста (из электронной почты, SMS, API, браузера, OCR) — ответственность вызывающего кода;
`textseek` работает только с тем текстом, который вы ему передаёте.

```python
email_text = email.text + "\n" + email.html
code = TextSeek(email_text).extract_one(Extract.code(length=6))
```
